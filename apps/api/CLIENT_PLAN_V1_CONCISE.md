# AI Video Automation Plan

## Objective

Deliver a practical Phase 1 system that increases output and consistency while controlling cost.

This phase focuses on:
- faster production workflow
- better quality control
- support for 20s and 30s video formats
- clear reporting foundation for sponsor value

---

## What Will Be Delivered First

1. **Configurable clip workflow**
   - 20s mode: 2 clips
   - 30s mode: 3 clips (new option)
   - default clip duration remains 10s for reliability

2. **Human checkpoint after first clip**
   - Clip 2/3 only runs after first clip is approved
   - reduces wasted generation cost

3. **End card pool support**
   - multiple end cards per client/sponsor
   - rotate selection instead of always using one fixed card

4. **Operator workflow improvements**
   - cleaner run flow for internal team
   - standardized job steps and run tracking

5. **Run tracking foundation**
   - capture clip settings, retries, failures, approvals, and output metadata
   - prepares for optimization and analytics in next phase

6. **Asset intake and storage flow**
   - continue using Google Drive for initial client asset collection (short term)
   - move approved assets into structured pipeline storage for production jobs
   - maintain consistent folder/schema standards for reliable automation

---

## What Is Not Included In This First Step

- full public client portal
- full automated posting to all social platforms
- advanced self-learning agent actions
- full cloud storage migration with enterprise lifecycle policies

These are Phase 2/3 items after production workflow is stable.

---

## Short Timeline (Fast Delivery)

## Week 1
- finalize workflow rules (20s/30s, approval gate, end card rotation)
- implement orchestration updates in pipeline

## Week 2
- implement review gate + 3rd clip loop + end card pool logic
- run internal test batch and validate outputs

## Week 3
- production pilot with selected clients
- tune settings and finalize handoff checklist

---

## Success Criteria

- team can produce both 20s and 30s videos reliably
- first-clip approval gate is enforced
- end card rotation works as configured
- measurable reduction in failed/wasted generations
- pilot runs are approved for regular production use
